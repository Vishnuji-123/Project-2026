# 📊 About Firstrade

**Firstrade** is a **demo trading analytics platform** designed to connect with the **MetaTrader 5 (MT5) API** and provide deep insights into trading performance.

The platform helps traders:

📈 Analyze trade performance  
📊 Visualize trading metrics  
⚡ Track profits and losses  
📉 Monitor risk exposure  
📡 Integrate real trading data through MT5  

This project demonstrates **API integration, financial data processing, and trading analytics dashboards**.
