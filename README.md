## 📊 About the Project

**Firstrade** is a demo trading analytics platform that connects with the **MetaTrader 5 (MT5) API** to analyze trading activity and provide insights for traders.

It helps users:
- Analyze trades
- Track performance
- Visualize trading metrics
- Understand risk and profitability

This project is built for learning **trading analytics, API integration, and financial data visualization.**

---

## ✨ Features

✔ Connects with MT5 API  
✔ Fetch live/demo trading data  
✔ Trade analytics dashboard  
✔ Performance metrics visualization

✔ Interactive charts and insights
